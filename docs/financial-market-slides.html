HƯỚNG TRIỂN KHAI · 02 / 03

Anomaly Detection System

Phát hiện giao dịch bất thường (pump & dump, wash trading, fat-finger) theo thời gian thực. Dùng statistical model + ML trên stream data.

TECH STACK

Kafka Apache Flink Isolation Forest MLflow ClickHouse Grafana Prefect

ĐIỂM KHÓ NHẤT

Phân biệt "giao dịch bất thường thật" vs "biến động thị trường bình thường" — đòi hỏi feature engineering tinh tế và threshold tuning liên tục.

01 Detect volume spike bất thường: so sánh với baseline 30-day rolling avg
02 Price velocity alert: giá tăng/giảm quá X% trong dưới 60 giây
03 Order book imbalance detection: buy/sell pressure ratio
04 ML scoring: Isolation Forest real-time inference trên Flink

Demo ấn tượng

Replay historical data của một sự kiện thao túng giá thật → pipeline detect được anomaly với timestamp và confidence score. Compelling case study.

Thách thức Mô tả Mức độ
